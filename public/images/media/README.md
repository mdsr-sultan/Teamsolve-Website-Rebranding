# Media Images

This folder contains images for LinkedIn posts displayed on the Media page.

## Fallback Images

The following images are used as fallbacks when LinkedIn API is not configured:

- `breakfast-bowl.jpg` - Food manufacturing article
- `ai-water-solutions.jpg` - AI water solutions partnership
- `balibago-waterworks.jpg` - Philippines customer announcement
- `wwa-singapore.jpg` - WWA Knowledge Twin implementation
- `placeholder.jpg` - Default placeholder image

## Notes

- Images should be optimized for web (< 500KB recommended)
- Recommended dimensions: 1200x630px (LinkedIn share format)
- Supported formats: JPG, PNG, WebP
